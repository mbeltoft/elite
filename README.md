# Elite

**Animate spaceships from Elite**

This repository builds on tsoding's "One Formula That Demystifies 3D Graphics"
repository https://github.com/tsoding/formula.

The video behind the repository explains how very basic 3D projection onto a
2D screen works.

I wanted to view the wireframe spaceships from the Elite space computer game,
which I found here: https://elite.bbcelite.com/deep_dives/ship_blueprints.html

The format was not directly usable, but after converting into arrays of points
and edges I added scaling of the points in the models to the "unit" world and
the ship models displayed fine.
